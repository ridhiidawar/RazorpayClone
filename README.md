# Razorpay Clone 💳

A fully responsive Razorpay homepage clone using **HTML**, **Tailwind CSS (via CDN)**, and custom styling.

## 🚀 Features
- Responsive layout
- Tailwind CDN used (no npm setup)
- Smooth transitions & modern UI

## 📁 Files Included
- `index.html` – Main page
- `assets/` – Icons & images
